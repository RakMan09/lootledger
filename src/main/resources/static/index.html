◈ LootLedger

A dupe-proof, double-entry game economy. Gold and items can never be duplicated or lost — no matter how clients retry, disconnect, or double-click. Try to break it below.

API docs (Swagger) Live invariant check Source on GitHub

🔨 Try to dupe the economy

Fire many identical transfer requests at the exact same moment, all sharing one Idempotency-Key — exactly what a laggy client or a double-click attack does. A naive system would apply them all and duplicate gold. LootLedger applies it exactly once.
